Multi (Gang) Programmers

BeeHive304

BeeHive304는 경쟁모델 중 가장 빠른 64 핀드라브 초고속 범용 멀티롬라이터입니다. 2개의 독립적인 art FPGA 설계로 강력한 ARM 프로세서와 내부 SSD를 장착하여 초당 31MB 프로그램속도를 자랑하며, 4개의 BeeProg3 모듈이 독립적으로 장착된 멀티형으로 고용량 메모리의 양산 프로그램에 적합하고 정교하게 설계되어 품질과 신뢰성이 매우 뛰어난 모델입니다. 기존 DIL소켓변환 콘버터 방식이 아닌, 보다 확실한 접점과 고속프로그램에 맞는 Programming module 방식을 채택합니다.

BeeHive208S

BeeHive208S는 8x 48 핀드라브를 장착한 Stand-alone 범용 멀티 롬라이터입니다. 강력한 FPGA 설계로 초당 5MB 프로그래밍 속도를 자랑하며, 품질과 신뢰성이 뛰어난 모델입니다. 또한, ISP 프로그램을 지원하고, 별도의 PC 연결없이 LCD 터치 스크린으로 모든 작업이 가능합니다. 8개의 BeeProg2 모듈이 독립적으로 장착된 멀티형으로 각 모듈은 BeeProg2 성능과 같으며, Pg4uwMC환경을 사용합니다. eMMC, NAND, UFS 등 매우 큰 사이즈 메모리의 경우 BeeHive304 모델을 권장합니다.

BeeHive204AP-AU

BeeHive204AP-AU는 4x 48 핀드라브를 장착한 범용 멀티 롬라이터입니다. 강력한 FPGA 설계로 초당 5MB 프로그래밍 속도를 자랑하며, 품질과 신뢰성이 뛰어난 모델입니다. 자동 ZIF socket opener를 장착하여 프로그램 완료 후 소켓이 자동으로 Open 되어 작업성이 편리하고 생산성 뛰어나고, 손가락의 피로도가 없습니다. 또한, 오차없이 소켓이 구동되므로 소켓의 수명연장에 도움이 됩니다. 4개의 BeeProg2 모듈이 독립적으로 장착된 멀티형으로 각 모듈은 BeeProg2 성능과 같으며, Pg4uwMC환경을 사용합니다.

BeeHive204

BeeHive204는 4x 48 핀드라브를 장착한 범용 멀티 롬라이터입니다. 강력한 FPGA 설계로 초당 5MB 프로그래밍 속도를 자랑하며, 품질과 신뢰성이 뛰어난 모델입니다. 또한, ISP 프로그램을 지원하여 SMT후 후공정프로그램이 가능합니다. 4개의 BeeProg2 모듈이 독립적으로 장착된 멀티형으로 각 모듈은 BeeProg2 성능과 같으며, Pg4uwMC환경을 사용합니다. 거의 모든 칩을 지원하는 400만원대의 양산용 갱롬라이터로 가성비가 매우 우수합니다. eMMC, NAND, UFS 등 매우 큰 사이즈 메모리의 경우 BeeHive304 모델을 추천합니다.

Single Programmers

BeeProg3

BeeProg3는 64 핀드라브 초고속 범용 롬라이터입니다. 2개의 독립적인 art FPGA 설계로 강력한 ARM 프로세서와 내부 SSD를 장착하여 초당 31MB 프로그램속도를 자랑하며, 고용량 메모리 프로그램에 적합하고 정교하게 설계되어 품질과 신뢰성이 매우 뛰어난 모델입니다. 기존 DIL소켓변환 콘버터 방식이 아닌, 보다 확실한 접점과 고속프로그램에 맞는 Programming module 방식을 채택합니다.

BeeProg2

BeeProg2는 48 핀드라브를 장착한 엘넥사의 대표적인 범용 롬라이터입니다. 강력한 FPGA 설계로 초당 5MB 프로그래밍 속도를 자랑하며, 품질과 신뢰성이 뛰어난 모델입니다. 또한, ISP 프로그램을 지원하여 SMT후 후공정 프로그래밍이 가능합니다. 보다 저렴한 모델을 보시려면 BeeProg2C를 검토해 보기 바랍니다

BeeProg2C

BeeProg2C는 48 핀드라브를 장착한 엘넥사의 경제적인 가격의 범용 롬라이터입니다. BeeProg2C는 BeeProg2와 대부분의 사양이 같지만, 지원 디바이스가 일부 제한적이며, LPT parallel printer 포트가 없습니다. 보다 많고 다양한 디바이스를 지원하는 모델을 보시려면 BeeProg2를 검토해 보기 바랍니다.
